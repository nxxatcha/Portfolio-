Statement of purpose


เหตุผลที่อยากเข้าศึกษาต่อ ในคณะวิศวกรรมศาตร์ สาขาคอมพิวเตอร์ มหาวิทยาลัยเกษตรศาสตร์บางเขน


        เหตุผลที่หนูอยากเข้าศึกษาต่อในคณะวิศวกรรมคอมพิวเตอร์ คือ หนูรู้สึกว่าในการทำงานด้านนี้ หนูทำได้ดี หนูได้เริ่มเขียนโค้ดตอนช่วงม.1ค่ะ ตอนนั้นโรงเรียนส่งตัวไปแข่ง เป็นการแข่งเขียน Webpage ประเภท text editor ตอนนั้นได้เหรียญทองแดงค่ะ เพราะยังไม่ค่อยเข้าใจหลักการทำงานเท่าไหร่ แต่พอขึ้นม.2 ไปแข่งอีกรอบก็ได้ รองชนะเลิศอันดับ1 เหรียญทอง ตอนนั้นทำให้หนูรู้สึกสนใจที่จะเรียนต่อทางด้าน IT และ คอมพิวเตอร์ค่ะ และพอช่วงม.5หลังโควิดหนูย้ายโรงเรียน หนูก็ได้พยายามพาตัวเองไปอยู่ในจุดที่สามารถต่อยอดทักษะทางด้านคอมพิวเตอร์ เช่นการเลือกเรียนวิชาศักยภาพ คอมพิวเตอร์ เป็นการเรียนเนื้อหาต่างๆเกี่ยวกับคอมพิวเตอร์ เช่นการใช้ภาษา JAVA และก็ได้มีโอกาสไปแข่งขัน เขียน Web application และได้รางวัลเหรียญเงินค่ะ

        และในช่วงม.5 นั้นหนูก็ได้มีโอกาสได้ไปเข้าร่วมค่าย Let me tried ขอใ มหาวิทยาลัยเกษตรศาสตร์ ได้ฟังหลักสูตร ว่ามีการลดวิชาเรียนอย่างแคลคูลัสไป ภายในค่ายพี่ๆ สอนเรื่งการเขียน htmlแล้วก็ใช้ css ควบคู่กันไปค่ะ ทำให้หนูสามารถสร้าเว็บไซต์เล็กๆนี้ขึ้นมาได้ และบรรยากาศภายในม.ก็ร่มรื่นย์น่าอยู่ และตัวตึกของคณะวิศวคอมก็ดูเป็นส่วนตัวดีค่ะ และในท้ายค่าย ก็มีพี่ๆที่เรียนจบไปแล้ว หรือไม่ก็กำลังฝึกงานอยู่ มาเล่าให้ฟังว่า จบไปแล้วทำงานด้านไหนต่อ ซึ่งมันทำให้หนูรู้สึกอยากเข้าเรียนที่ ม.เกษตรนี้มากๆเเลยค่ะ

        สุดท้ายนี้ หนูเชื่อว่าถ้าคณะกรรมการรับเลือกหนูเข้าศึกษาต่อ หนูจะไม่ทำให้ผิดหวังค่ะ ถึงหนูจะไม่ใช่คนที่หัวดีมาก และหนูก็จะพยายามเรียนรู้ให้ได้มากที่สุดค่ะ หนูอยากเข้าที่นี่จริงๆค่ะ ขอบคุณค่ะ
